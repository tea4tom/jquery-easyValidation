easyValidation.js

A quick and easy plugin for validing input fields on forms in real-time.

Drop into existing projects by including easyValidation.css inbetween your HTML <head> tags and include the script(s) jquery* & easyValidation.js at the end of your content before the </body> tag.

Any forms you want to highlight user mistakes on need to have a class specified "easyValidate" to be included and you can control the type of validation by adding a data attribute "data-validate" to your text fields:

  • data-validate="notnull"
    Makes sure the user does not leave the field blank.
  • data-validate="email"
    Compares user input to a valid email regular expresssion to make sure there is a username / domain name and an @ symbol.
  • data-validate="telephone"
    Compares the users input to a regular expression that closely matches a phone number.

* The jquery framework is required for this project. It has been developed using version 3.3.0 so make sure your current jquery API is compatible.
